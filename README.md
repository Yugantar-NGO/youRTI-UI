# youRTI - RTI Transparency Dashboard

A citizen-focused platform that makes Right to Information (RTI) data in India accessible, understandable, and actionable.

## About

youRTI is built by **Yugantar NGO** to promote government transparency and accountability. The platform tracks RTI requests across India, highlighting both successes and areas needing attention.

## What It Does

- **Tracks RTI outcomes** - See which requests led to information disclosure, policy changes, or accountability actions
- **Highlights impact stories** - Real examples of how RTI has made a difference for citizens
- **Shows pending cases** - Brings attention to unanswered requests that need follow-up
- **Provides insights** - Understand trends in government responsiveness across departments and regions

## Who It's For

- **Citizens** seeking transparency in government processes
- **Journalists** investigating accountability stories
- **Researchers** studying governance and public administration
- **Activists** working on transparency and anti-corruption efforts

## Features

- Daily edition format with curated RTI stories
- Department-wise and state-wise performance breakdowns
- Impact metrics showing the real-world effects of RTI
- Browse by topics like healthcare, education, infrastructure, and more
- Activity feed tracking the latest RTI developments

## Getting Started

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
